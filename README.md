# 🧠 Sentiment Analyser Web App

> Live BERT-based sentiment analysis web application built with Streamlit.

## Live Demo
Try the app: [Sentiment Analyser](https://cannot-turban-pretext.ngrok-free.dev)

## What it does
- Analyses text as POSITIVE or NEGATIVE
- Shows confidence score as a percentage
- Displays a live gauge chart
- Bulk analysis — analyse multiple texts at once
- Summary statistics for batch analysis

## Technologies used
- Python 3.11
- HuggingFace Transformers (DistilBERT)
- Streamlit
- Plotly
- Google Colab

## How to run
1. Open sentiment_analyser.ipynb in Google Colab
2. Run all cells in order
3. Copy the ngrok URL that appears
4. Open in any browser — app is live!

## Model
- DistilBERT fine-tuned on SST-2 dataset
- Binary classification: POSITIVE / NEGATIVE
- Confidence score with each prediction
